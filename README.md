## 🎓 O projeto

O projeto de PI do semestre que deu origem a este jogo foi fazer uma cutscene de um jogo, e no caso tivemos a ideia de fazer o escopo em um jogo focado para celulares feito na Unity.

## 👨🏽‍💻 Tecnologias

* C#
* Unity
* Pacote Adobe
* Audacity
* Texture Pack
* Mixcraft
